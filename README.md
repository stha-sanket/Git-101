# Git 101 🚀

A beginner-friendly repository to learn Git step by step through practical examples and real commit history.

---

## 📌 What is Git?

Git is a **version control system** that helps you:

* Track changes in your code
* Collaborate with others
* Revert to previous versions when needed

---

## 🧠 What You’ll Learn

* How Git works
* Common Git commands
* Branching and merging
* Writing clean commit messages
* Basic project versioning

---

## 🌱 Getting Started

1. Clone the repository:

```bash
git clone 
```

2. Navigate into the folder:

```bash
cd git-101 https://github.com/stha-sanket/Git-101/
```

3. Start exploring and making commits 🚀

---

## 📝 Commit Message Style

Use clear and meaningful messages:

* `feat: add new feature`
* `fix: resolve bug`
* `docs: update README`
* `chore: cleanup files`
